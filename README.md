# folder_mapper

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

Output: 
```
├── node_modules
│   ├── .bin
│   │   ├── tsc.bunx
│   │   ├── tsc.exe
│   │   ├── tsserver.bunx
│   │   └── tsserver.exe
│   └── @types
│       └── bun
│           ├── LICENSE
│           ├── README.md
│           ├── index.d.ts
│           └── package.json
├── .gitignore
├── bun.lock
├── index.ts
└── temp.ts
```
